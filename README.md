# Moonshine Capital — Affiliate Link Generator & Tracker

Internal partner operations app for generating, organizing, tracking, and reconciling affiliate links.

## Stack
- Next.js 14 App Router
- TypeScript
- TailwindCSS
- TanStack Table
- Recharts
- Zustand
- React Hook Form + Zod

## Run locally
```bash
npm install
npm run dev
```

Open `http://localhost:3000/dashboard`

## Deploy
1. Push this repo to GitHub
2. Import the repo into Vercel
3. Deploy

This repo is mock-data powered, adapter-backed, and ready for future backend integration.
